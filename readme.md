## Getting Started

**1. Clone the Repository**

```bash
https://github.com/wawrzonnn/panini-creator.git
```

**2. Install Dependencies**

```bash
npm install
```

**3. Run the Application**

```bash
npm run dev
```

The application will be running on **http://localhost:5173.**
